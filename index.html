Vue.js - Les bases

Instructions

  1. Chargez Vue.js (il est dans le dossier js/), et affecter une vue à l'élément d'identifiant monapp.
  2. Ajoutez une propriété message valant "Salut" au modèle de données Vue.
  3. Affichez le contenu de message dans le premier paragraphe en utilisant la directive v-text.
  4. Affichez le contenu de message dans le deuxième paragraphe avec la syntaxe « moustache ».
  5. Ajoutez une propriété dateJour initialisée à la date du jour, et affichez l'année courante dans le troisième paragraphe
    Les dates en JavaScript
  6. Créer une propriété lienJu initialisée avec https://www.jura.ch
    Modifier l'attribut href du lien du quatrième paragraphe pour qu'il pointe vers lienJu.
