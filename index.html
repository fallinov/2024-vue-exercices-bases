Vue.js - Les bases

Instructions

  1. Chargez Vue.js dans cette page HTML, il est dans le dossier js/
  2. Créer une nouvelle application Vue et la monter sur l'élément HTML avec l'id #monapp.
  3. Ajoutez une donnée message à votre Vue avec comme valeur 'Salut'.
  4. Affichez le contenu de message dans le premier paragraphe en utilisant la directive v-text.
  5. Affichez le contenu de message dans le deuxième paragraphe avec la syntaxe « moustache » {{ moustâââche }}.
  6. Ajoutez une propriété dateJour initialisée à la date du jour, et affichez l'année courante dans le troisième paragraphe. Comprendre la date et l'heure en JavaScript
  7. Ajouter une donnée lienJu à votre Vue initialisée avec 'https://www.jura.ch'
    Modifier l'attribut href du lien du quatrième paragraphe pour qu'il pointe vers le contenu de la donnée lienJu.
